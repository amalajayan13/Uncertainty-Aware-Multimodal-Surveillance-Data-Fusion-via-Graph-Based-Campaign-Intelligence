# Uncertainty-Aware-Multimodal-Surveillance-Data-Fusion-via-Graph-Based-Campaign-Intelligence

## Reproducibility
This repository contains the Jupyter notebook used for the analysis in the preparation of the manuscript. The notebook `mainv2.ipynb` can be opened directly in Google Colab or local machine. The .yaml file `conda-env.yaml` contains the conda environment configuration used to run the script on the local machine using Anaconda. Change the paths accordingly.
